# Car Cleaning
A sample app to practice using Xamarin Forms by creating an app to manage cleaning tasks for a Car Cleaning Company

## Highlights
- Xamarin Forms 4.5
- Reading from embedded JSON file
- Created a calendar control
- Collection View
- IValueConverters
- MVVM
- Visual State Manager
